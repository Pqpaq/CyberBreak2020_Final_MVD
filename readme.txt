В папке windows находится версия приложения по windows.
В папке linux находится версия приложения для linux.
Чтобы собрать приложение для linux нужно взяь всё содержимое папки linux и собрать его в deb пакет.
В папке converter находится скомпилированная версия программы для windows 10. Для запуска нужно запустить converter.exe
